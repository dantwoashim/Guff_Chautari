# Persona Creation Guide

## Paths
- Create persona in-app using persona setup workflows.
- Import persona using `.persona` YAML.

## Recommended `.persona` fields
- Identity: name, role, boundaries.
- Communication: tone and speaking style.
- Behavior: priorities and decision style.
- Safety: refusal and escalation conditions.

## Import Steps
1. Open Persona Import panel.
2. Paste or upload YAML.
3. Validate schema.
4. Preview generated profile.
5. Confirm import.

## Quality Checklist
- Keep persona prompts concise.
- Avoid contradictory rules.
- Define explicit boundaries.
